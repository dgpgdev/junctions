# MATH

## add
add two number
#### parameters
| name | type | description |
|------|------|-------------|
|x|Number|first number|
|y|Number|second number|

<span class='return'>Return ></span> Number
```js
const result = add(10,2) //return 12
const result = add(10)(2) //return 12
const y = add(10)
const result = y(2)//return 12
```
